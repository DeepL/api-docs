The [DeepL API Status Page](https://api-status.deepl.com) provides real-time information about the operational status of the DeepL API.

## What you can check

- **Current status** of all API services across Europe, Americas, and Asia-Pacific regions
- **90-day availability** history per service group (Translation, Write, Document Translation, Glossaries, and more)
- **Active and past incidents** affecting the API
- **Pro and Free API tier** status separately

## When to check

- If you're receiving HTTP 5xx errors from the API
- If you're experiencing elevated latency or timeouts
- Before investigating issues in your own integration — confirm the API is operational first

## Regions

The DeepL API operates in three regions. Each region has its own endpoint and status:

| Region | API Endpoint |
|--------|-------------|
| Europe | `api.deepl.com` |
| Americas | `api-us.deepl.com` |
| Asia-Pacific | `api-jp.deepl.com` |
