---
openapi: post /v3/glossaries
title: "Create a glossary"
description: "Create a multilingual glossary containing one or more language-pair dictionaries using the v3 endpoint."
---

Creates a glossary containing one or more dictionaries. Each dictionary maps source phrases to target phrases for a single language pair. A single glossary can contain dictionaries for multiple language pairs.

See the [Glossaries overview](/api-reference/multilingual-glossaries) for entry formats, how to use a glossary in a translation request, and details on editing, deleting, and language support.
