---
openapi: post /v2/translate
title: "Translate text"
description: "Translate one or more texts into a target language, with optional glossary, style, formality, and tag-handling controls."
---

Translates up to 50 texts per request into the specified target language. Source language detection is automatic when `source_lang` is omitted.

See the [Translate text overview](/api-reference/translate) for request body descriptions, model selection guidance, sentence splitting behavior, tag handling, and examples for translating large volumes of text.
